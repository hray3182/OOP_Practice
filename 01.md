# 練習1

建立一個 Character 的 Class，需包含以下屬性及方法

## 屬性:

* name

* position_x

* position_y

## 方法:

* \_\_str\_\_(): 返回角色的名字及位置

* move_left: 向左走一格

* move_right: 向右走一格

* move_up: 向上走一格

* move_down: 向下走一格

## 任務

1. 創建一個 character 物件，名字叫做"Byron"，位置在 x:0, y:0

2. 印出角色的名字及位置

3. 讓他走到 x:3, y:5

4. 重新印出角色的名字及位置

5. 讓他走到 x:-5, y:-9

6. 重新印出角色的名字及位置
